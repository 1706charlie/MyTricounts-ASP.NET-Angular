# Application Tricount (🚧 Projet en développement actif)

Reproduction moderne de l'application Tricount, développée en full stack avec  
un backend complet en ASP.NET Core et un frontend Angular.

---

## ✅ Backend — Terminé
Backend entièrement fonctionnel : API REST, Entity Framework Core, règles métiers, sécurité, authentification JWT, gestion complète des tricounts, opérations et utilisateurs.

---

## 🧩 Frontend — Terminé
Fonctionnalités finalisées :
- Login / Logout / Signup  
- Reset database  
- Gestion du token JWT  
- Layout, navigation, guards de routes  

---

## 🧩 Frontend — En refactorisation / développement
- Module Tricounts : affichage des données, actions utilisateur, organisation du code
